# JCalc

A Java Swing calculator with two modes — a clean simple calculator and a full-featured classic calculator.

![Preview1](https://github.com/DavoDC/JCalc/blob/master/Preview1.png)
![Preview2](https://github.com/DavoDC/JCalc/blob/master/Preview2.png)

## Features
- **Simple mode** — clean, minimal interface for everyday calculations
- **Classic mode** — full calculator with extended functionality
- Switch between modes on the fly
- Nimbus look and feel

## Usage
Run the executable from `Executable/`. Starts in simple mode — switch to classic from the menu.

## Tech
- Java (Swing / Nimbus L&F)
- Built with Launch4J for Windows executable packaging
